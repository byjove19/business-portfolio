# ByJove Technologies Portfolio

This is a portfolio template project designed for ByJove Technologies. It serves as a showcase of my development work, built using modern web technologies. The project is fully responsive, featuring a clean and engaging user interface.

## Features
- Responsive design suitable for both desktop and mobile views.
- Interactive social media links with icons (Email, LinkedIn, Twitter, Facebook, Instagram).
- Contact information and professional details displayed in a visually appealing layout.
- A section for downloading the CV (resume) and viewing the personal introduction.

## Technologies Used

- **HTML5**: Structure and content of the website.
- **CSS3**: Styling for layout, colors, typography, and responsiveness.
  - **CSS Variables** for easier theming and spacing management.
- **SVG Icons**: Used for social media links and other visual elements to enhance the user interface.
- **JavaScript**: For dynamic content manipulation and event handling, enhancing the user experience.
- **Responsive Design**: The website is built to be mobile-first, ensuring an optimized experience across various devices.
